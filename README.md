# Sistema_Imobili-ria_C-
Sistema de gestão imobiliária em C# para controle de inventário e cálculo automático de comissões por metragem. Gerencie cadastros, status de venda (Disponível, Vendido, Alugado) e armazene dados localmente em arquivos de texto. Solução simples e eficiente para automação de processos imobiliários.
